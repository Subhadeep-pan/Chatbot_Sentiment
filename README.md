# ChatBot with Sentiment Analysis
## Objective
This project implements a simple chatbot that:
- Maintains full conversation history.
- Performs sentiment analysis at:
  - Conversation level (Tier 1 – mandatory).
  - Statement level for each user message (Tier 2 – additional credit).
- Optionally summarises mood trend across the conversation.
## Technologies Used
- Language: Python 3
- Sentiment Analysis Library: [TextBlob](https://textblob.readthedocs.io/)
- Environment: CLI (Command Line Interface)
## Setup & How to Run
